Namespace

docker基于namespace进行进程隔离

docker容器只是运行了一个独立的进程，这个进程使用namespace实现与其他namespace、宿主机。

隔离模式

  • pid
  • uts
  • mount
  • user
  • network
  • ipc

那么docker容器就是运行了一个由多个隔离限制组合起来的进程。

与虚拟机不同的是，容器只是启动了一个特殊的进程，使用的还是宿主机的操作系统内核，与宿主机共享资源。
